20 Raccourcis Utiles dans Visual Studio Code

Commande Raccourci Description
Ouvrir un fichier Ctrl + O Ouvre un fichier existant.
Enregistrer Ctrl + S Enregistre le fichier actif.
Couper Ctrl + X Coupe le texte sélectionné.
Copier Ctrl + C Copie le texte sélectionné.
Coller Ctrl + V Colle le texte depuis le presse-papiers.
Annuler Ctrl + Z Annule la dernière action.
Refaire Ctrl + Y Refait la dernière action annulée.
Rechercher Ctrl + F Recherche du texte dans le fichier actuel.
Remplacer Ctrl + H Remplace du texte dans le fichier actuel.
Rechercher dans tout le projet Ctrl + Maj + F Recherche du texte dans tous les fichiers du projet.
Ouvrir le terminal Ctrl + ` Ouvre le terminal intégré.
Commenter une ligne Ctrl + : Ajoute un commentaire à la ligne sélectionnée.
Formater le code Maj + Alt + F Formate le code selon les règles définies.
Basculer le panneau latéral Ctrl + B Masque ou affiche le panneau latéral.
Explorer les fichiers Ctrl + P Ouvre rapidement les fichiers.
Aller à la ligne Ctrl + G Déplace le curseur à une ligne spécifique.
Aller à la définition F12 Va à la définition de la fonction ou variable.
Ouvrir les paramètres Ctrl + , Ouvre la fenêtre de paramètres.
Afficher l'aperçu Markdown Ctrl + Maj + V Affiche un aperçu du fichier Markdown.
Renommer la variable F2 Renomme toutes les instances d'une variable.
